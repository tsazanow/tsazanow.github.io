Обо мне

Здравствуйте! Меня зовут Артем и я html-верстальщик.

Верстаю адаптивные и фиксированные сайты по макетам, используя современные инструменты и технологии. Прошел профессиональные курсы по HTML и CSS в одной из лучших онлайн-школ по веб-разработке HTML Academy. В ходе учебы получил ценный опыт в профессии работая с опытными наставниками и преподавателями.
По итогам учебы научился применять ряд незаменимых в современной веб-разработке технологий и методик.

Люблю узнавать новое и применять изученное в последующих проектах. Постоянно учусь и совершенствуюсь в области веб-разработки.

Опыт и инструменты

В мире веб-разработки есть множество инструментов для быстрого и качественного создания сайтов. Я слежу за их развитием и стараюсь применять на практике

HTML

Стараюсь делать разметку семантично, применяя атрибуты и теги согласно спецификациям HTML.

CSS

Новые проекты верстаю на flexbox.

Методология БЭМ

Для простоты написания и поддержки кода использую методологию именования классов БЭМ.

SVG-графика

В проектах, где это необходимо и целесообразно применяю векторную графику.

Системы контроля версий

Использую в работе преимущества системы контроля версий Git для удобной работы с репозиториями своих проектов, комментирования и отправки изменений.

Bootstrap

Могу использовать в некоторых проектах готовые компоненты CSS-фреймворка Bootstrap(например сетку).

Адаптивная верстка

Верстаю страницы адаптивно и кроссбраузерно. Использую различные графические редакторы - Photoshop, Figma, Adobe XD, Avocode.
