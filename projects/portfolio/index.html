обо мне

Артём Сазанов

Верстальщик, фрилансер. Постоянно учусь и совершенствуюсь в области веб-разработки.

Верстаю адаптивные и фиксированные сайты по макетам, используя современные инструменты и технологии. Люблю узнавать новое и применять изученное в последующих проектах.

Прошел профессиональные курсы по HTML и CSS в одной из лучших онлайн-школ по веб-разработке HTML Academy. В ходе учебы получил ценный опыт в профессии работая с опытными наставниками и преподавателями.
По итогам учебы научился применять ряд незаменимых в современной веб-разработке технологий и методик.

Скиллы

В мире веб-разработки есть множество инструментов для быстрого и качественного создания сайтов. Я слежу за их развитием и стараюсь применять на практике

HTML

Научился делать разметку семантично, применяя атрибуты и теги согласно спецификациям HTML.

CSS

Изучил стилевой язык CSS. Новые проекты верстаю на flexbox.

Научился использовать преимущества системы контроля версий Git для удобной работы с удаленными репозиториями, комментирования и отправки изменений.

Bootstrap

Освоил CSS-фреймворк Bootstrap для применения его готовой сетки в некоторых проектах.

Адаптивная верстка

Препроцессоры
